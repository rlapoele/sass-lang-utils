    SCSS-LANG-UTILS

    > A small library of commonly used sass functions written in SCSS.

    Motivation

    After writing several libraries, frameworks and style generators in scss for different projects I worked on, I realised that I often ended-up re-searching and re-writing many sass/scss language related utility functions.

    I therefore decided to bundle these into a small library and make it available to everyone.

    Documentation has been generated using the excellent sassdoc tool and is available in HTML format in the /docs folder.

    slu-list-every

    Since v1.0.0
    @function slu-list-every($list, $predicate-function-name, $predicate-function-args...) { ... }
    View source

    Description

    Tests each list element against a predicate and returns true if the predicate returns true for every elements.

    In order to pass a list item as argument of the every function, one must include 'item' with or without quote as one of the $predicate-function-args argument.

    The same logic applies to list item index where one must include 'index' or 'item-index' or, finally, 'list-item-index'.

    Lastly, a similar approach can be taken to pass the list itself; one can use 'list' or 'collection'.

    Any other required arguments can also be passed and positioned at any needed place.

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $list

    the list of item to be processed

    		List— none
    $predicate-function-name

    the predicate name to be called for each list item

    		String— none
    $predicate-function-args

    arguments to be passed to the predicate function

    		Arglist— none

    Returns

    Bool —

    true if the predicate returns true for every list elements, false otherwise.

    Example

    slu-list-every function:

    $numbers: 1, 2, 3, 4;

@function greater-than($x, $y) {
  @return $x > $y;
}

$all-are-greater-than-zero: slu-list-every($numbers, greater-than, item, 0);

@debug inspect($all-are-greater-than-zero); // terminal/console output: true


$all-are-greater-than-three: slu-list-every($numbers, greater-than, item, 3);

@debug inspect($all-are-greater-than-three); // terminal/console output: false
